## 为什么要使用「等等标签」？

在日常电脑工作时，我们常常会不自觉地打开社交媒体网站、娱乐网站等让我们分心的网站，然后忘记了我们手头上真正重要的正事。又或者你正在试图减少使用社交媒体的时间，但又无法抗拒诱惑。

安装「等等标签」后，你可以设置指定的网址列表，当你访问这些网站时，会首先弹出提示你专注的页面，提醒你也许你应该先完成手头上的工作。

## 两种不同的提醒模式

### 🕜 倒计时模式

访问指定网站时，会弹出一个倒计时页面，倒计时结束后，你才能决定是否继续访问这个网站。倒计时提示文字可以自定义，你**可以利用自定义的文字来培养你自己希望养成的小习惯**。

### 🧘 冥想模式

访问指定网站时，会弹出一个引导你进行冥想专注练习的页面。你可以先进行冥想专注练习，然后再决定是否继续访问这个网站。

冥想时长会被记录下来，你可以在设置页面查看你累计的冥想时长。这种模式在阻止分心的同时，也可以**帮助你养成日常冥想的好习惯**。

## 常见问题

### 我的浏览记录会被上传吗？

不会。插件使用 chrome 的 tabs API 来获取当前打开的标签页的 URL，这些数据只会在你的本地计算机上使用，不会上传到任何服务器。插件也没有申请任何能获取你个人标识的权限。
